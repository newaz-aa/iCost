
Supplementary Materials

This package contains curated supplementary Excel workbooks for the manuscript:
'iCost: A Novel Instance-Complexity-Based Cost-Sensitive Learning Framework'.

Files:
1. Supplementary_File_S1_Binary_Default_Results.xlsx
   - Binary default-setting summaries, dataset information, win-loss analysis, error trade-off, stability/runtime, and raw confusion-count comparisons.

2. Supplementary_File_S2_Binary_Cost_Tuning_Results.xlsx
   - Cost-parameter search space, tuned MCC summaries, tuned iCost vs tuned CSL win-loss analysis, cost-value frequencies, and dataset-wise tuning comparisons.

3. Supplementary_File_S3_Resampling_Comparison.xlsx
   - Resampling comparison results, including raw long-format data and metric-wise sheets.

4. Supplementary_File_S4_Multiclass_Results.xlsx
   - Final multiclass manuscript summaries, and multiclass win-loss summaries.

Notes:
- MCC/F1 values in manuscript-style summary tables are reported on a 0-100 scale.
